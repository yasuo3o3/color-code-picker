# 🎨 Color Code Picker (カラーコードピッカー)

Chromeブラウザで、画面上の好きな場所の「色」をワンクリック（またはキーボード）で取得できる拡張機能です。
取得した色は、自動的にクリップボードにコピーされます。

## ✨ 特徴

- 🔍 Chrome標準のルーペ機能で正確にピクセルを選択できます。
- ⌨️ **キーボードショートカット**対応！ボタンの「ホバー状態（マウスを乗せた時の色）」も取得可能です。
- ⚙️ お好みの形式（HEX、RGB、HSL）で色をコピーするように設定できます。

---

## 📥 インストール方法（開発者モード）

現在、この拡張機能はファイルから直接読み込む必要があります。

1. このリポジトリ（フォルダ全体）をパソコンの好きな場所にダウンロードします。
2. Chromeブラウザを開き、アドレスバーに `chrome://extensions/` と入力して移動します。
3. 画面右上の **「デベロッパー モード」** のスイッチをオンにします。
4. 画面左上の **「パッケージ化されていない拡張機能を読み込む」** ボタンをクリックします。
5. ダウンロードしたフォルダを選択すると、インストールが完了します！

---

## 🚀 使い方

### 1. マウス操作で使う（基本）
Chromeの右上にある、パレット（拡張機能）アイコンの中の「Color Code Picker」のアイコンをクリックします。
ルーペが起動するので、画面上の色を取りたい場所をクリックするだけです。

> **💡 ヒント:** 拡張機能のアイコンを「ピン留め」しておくと、いつでもワンクリックで使えて便利です！

### 2. キーボードで使う（ホバー状態の色を取る時）
リンクやボタンにマウスを乗せた時にフワッと変わる「ホバー状態の色」を取りたい時は、キーボードを使います。

1. 色を取りたい要素の上にマウスを乗せたまま動かさないでください。
2. キーボードで **`Ctrl + Shift + E`**（Macの場合は `Command + Shift + E`）を押します。
3. ルーペが起動するので、そのままクリックして色を取得します！

> ⚠️ **ショートカットが効かない場合：**
> 初回インストール時、ショートカットが自動で登録されないことがあります。
> その場合は、以下の手順で手動登録をお願いします。
> 1. Chromeで `chrome://extensions/shortcuts` を開く
> 2. 「Color Code Picker」を探す
> 3. 「Pick a color on the current page」の横にある鉛筆マークをクリック
> 4. キーボードで `Ctrl + Shift + E` を直接押して登録完了！

---

## ⚙️ 色の形式（フォーマット）を変えるには

取得した色のコード形式（例: `#FFFFFF` ではなく `rgb(255, 255, 255)` にしたい場合）は、設定から変更できます。

1. 拡張機能のアイコンを **右クリック** します。
2. **「オプション」** をクリックします。
3. 設定画面が開くので、好きな形式（HEX / RGB / HSL）を選んでください。自動で保存されます。

---

## 🚫 注意事項
* Chromeの設定画面や、新しく開いた空のタブ、拡張機能のストア画面（`chrome://` などのURL）では、セキュリティ上の理由でカラーピッカーを起動することができません。**普通のウェブサイト上**でお使いください。
